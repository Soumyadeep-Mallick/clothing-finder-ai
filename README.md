🛍️ AI Clothing Finder

An AI-powered clothing recommendation web app that aggregates products from Flipkart and Amazon based on user preferences.

---

🚀 Features
- 🔍 Multi-platform product search (Flipkart + Amazon)
- 🎯 Smart filtering (gender, style, color, budget)
- 🧍 Gender-specific UI (including innerwear & cup size)
- 🛒 Direct purchase links
- 🖥️ Clean grid-based UI (Myntra-inspired)

---

🛠️ Tech Stack
- Python
- Streamlit
- Selenium (Web Scraping)
- Pandas

---

▶️ Run Locally
pip install -r requirements.txt
streamlit run app.py
